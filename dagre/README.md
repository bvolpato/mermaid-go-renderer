# Dagre Go Port

This directory contains a Go translation of the original [dagre](https://github.com/dagrejs/dagre) library. 
Dagre is a JavaScript library that makes it easy to lay out directed graphs on the client side.

The original library is distributed under the MIT License by Chris Pettitt.
This Go port maintains equivalent logic and behavior to support server-side rendering of flowchart and state diagrams that are 100% visually conformant with MermaidJS.
